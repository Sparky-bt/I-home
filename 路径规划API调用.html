应用设计——"你在哪"APP¶

用户上传一张地点图片，APP接入计算机视觉功能，识别到图片的拍摄地点信息。接入高德api IP定位功能定位到图片拍摄地址，识别出具体地址后，启用路径规划api规划到此地的最佳路径，接着启用天气查询api和交通态势api，查找到此地的天气状况和通向此地的交通拥堵情况。¶

In [ ]:
